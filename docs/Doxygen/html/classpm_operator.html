Detailed Description

template<size_t S>
class pmOperator< S >

This interface forms the base for any operator acting on the pmTerms like constants (pmConstants), variables (pmVariables), fields (pmFields), or particle systems (pmParticle_system). The number of operands of the operator in question is controlled by the template parameter S.
