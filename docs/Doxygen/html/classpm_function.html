Detailed Description

This class represents an algebraic or differential equation with pmExpression-s on the lhs and rhs. The lhs must contain a pmVariable which is overwritten by the "evaluate" and "solve" member functions. When a pmFunction object is destroyed the lhs and rhs pmExpression-s are destroyed as well except if they are pmVariable or pmConstant objects. They are always managed by the pmWorkspace object.
