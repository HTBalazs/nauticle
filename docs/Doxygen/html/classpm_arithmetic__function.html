Detailed Description

template<Ari_fn_type ARI_TYPE, size_t S>
class pmArithmetic_function< ARI_TYPE, S >

This class implements the following operations for the expression tree: summation, subtraction, multiplication, division, power, term-by-term product for two operands furthermore summation and subtraction with one operand.
