Detailed Description

This class contains the smoothing kernel implementations for 1, 2 and 3 dimensions. An object of this class behaves as a smoohting kernel through the evaluate function. The type of the kernel to use is set by set_kernel_type() member function.

Kernel notations: dWlnnnnn 12345678

1: optional flag for derivative. 2: W (weight function). 3: can be one of the followings: p (polynomial), s (spline), e (exponential) 4: order of the polynomial or spline. It is zero for exponential kernel. 5: number of peices along the radius. 6: influence radius/smoothing radius. 7: Dimensions. 8: serial number.
