Detailed Description

This class represents the problem to solve. The contructor recieves the file name of the *.XML configuration file. It contains three objects:

  • function space: contains all the functions to solve
  • parameter space: this object holds the general constant values relating to the calculations but not come up in any of the functions.
  • particle system: if contains a particle cloud on which the functions are interpreted.
